sequenceDiagram
    participant User as Người dùng
    participant Blazor as WebBlazor (UI)
    participant API as GroupController
    participant Service as UsGroupService
    participant DB as SQL Server

    User->>Blazor: Nhập thông tin Nhóm & Lưu
    Blazor->>API: POST /api/Group (với JWT Token)
    API->>Service: Create(model)
    Service->>DB: Kiểm tra trùng tên Nhóm (Name)
    DB-->>Service: OK (Không trùng)
    Service->>Service: Bắt đầu Transaction
    loop Sinh ID duy nhất
        Service->>Service: Sinh mã ID Nhóm (3 ký tự)
        Service->>DB: Kiểm tra ID tồn tại
    end
    Service->>Service: Gán thông tin Audit (CreatedBy/At, RowStatus=1)
    Service->>DB: Insert UsGroups Table
    Service->>DB: Commit Transaction
    DB-->>Service: Success
    Service-->>API: Trả về UsGroupViewModel
    API-->>Blazor: 200 OK
    Blazor-->>User: Hiển thị thông báo & Refresh Table
