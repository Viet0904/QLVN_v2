sequenceDiagram
    participant U as Người dùng
    participant B as WebBlazor (UI)
    participant A as UserController
    participant S as UsUserService
    participant DB as SQL Server

    Note over U, DB: Quy trình Lấy Danh Sách Phân Trang
    U->>B: Truy cập trang Danh sách/Tìm kiếm
    B->>A: GET api/User/paginated?searchTerm=...&page=1
    A->>S: GetPaginated(request)
    S->>S: GetAll() Active Users
    S->>S: Filter searchTerm & Sort
    S->>S: Skip/Take (Phân trang)
    S-->>A: PaginatedResponse<UsUserViewModel>
    A-->>B: 200 OK
    B-->>U: Render danh sách lên Table

    Note over U, DB: Quy trình Xóa Mềm (Soft Delete)
    U->>B: Nhấn nút Xóa
    B->>A: DELETE api/User/{id}
    A->>S: Delete(id)
    S->>DB: SELECT User by Id
    DB-->>S: Entity found
    S->>DB: UPDATE RowStatus = 2 (Deleted)
    DB-->>S: Success
    S-->>A: true
    A-->>B: 204 No Content
    B->>B: Refresh Data Table
    B-->>U: Thông báo xóa thành công
